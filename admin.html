📋 Instructions de déploiement
  1. Téléchargez les fichiers modifiés ci-dessus
  2. Remplacez les fichiers correspondants dans votre dossier ecommerce-site/
  3. Ouvrez un terminal dans ce dossier et exécutez :
git add .
git commit -m "Mise à jour via dashboard admin"
git push origin main

GitHub Pages se mettra à jour automatiquement dans 1–3 minutes.
